# StashFlow Features Guide

## Navigation & Organization
*   **Adaptive Layouts:** Fully responsive design adapting to mobile (bottom navigation bar) and tablet/desktop (side navigation rail).
*   **Categories:**
    *   **Scenes:** Browse, sort, and filter video scenes. Options to display in List, Grid, or TikTok-style (infinite scroll) formats.
    *   **Performers:** Browse performers with circular thumbnail grids.
    *   **Studios:** Browse studios.
    *   **Tags:** Browse tags to find specific content.
    *   **Galleries:** Browse image galleries and image chapters.
*   **Sorting & Filtering:** Detailed bottom-sheet menus are available for filtering and sorting items by rating, date created, counts (scene/image/gallery), etc.

## Video Playback
*   **Interactive Video Player:** Supports robust playback with optional Picture-in-Picture (PiP) and background audio modes.
*   **Mini Player:** A persistent mini-player allows continued playback while browsing other pages.
*   **Auto-hiding Controls:** Player controls automatically hide during playback and reappear upon interaction.
*   **Scene Rating:** Includes a 5-star UI for rating scenes directly in the details page (1-100 range mapping).
*   **Smart Playlists:** "Play Next" strategy dynamically follows the current query sequence (sort/filter) if the manual queue is empty.
*   **Stream Prewarming:** Streams prewarm in the background to minimize first-play delays.
